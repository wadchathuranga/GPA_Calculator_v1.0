### About ###
This is a Grade Point Average (GPA) value calculator mobile application.

This app computes Semester GPA, Year GPA, current GPA and final GPA
based on course credits and the grade achieved.

This is a 4.0 scalar GPA Calculator

If your university computes GPA with different weights for each year separately
this app allows to enter the weight according to your university preference.

Developed by Dilshan Chathuranga.
Sabaragamuwa University of Sri Lanka.

Copyright @2022. All rights reserved.


### Weight ###
In here you can add weight values according to each four years
if you are not set the any value application will use default values (1/4) or (0.25)
for each year separately to compute the final GPA value.
